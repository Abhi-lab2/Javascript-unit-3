All about Objects

  • Objects are basically uses Dictionary type method to store the data
  • It uses Key-Value form to store the data
  • Everything in the JavaScript is in the form of Objects
  • Primitive values are immutable (they are hardcoded and therefore cannot be changed).
  • Example:-
    var person = { first_Name:"John", last_Name:"Pablo", age:20, eyeColor:"black" };
  • We can use object literals, as a new way to describing objects
