All about Strings

  • A string is zero or more characters written inside quotes - var str = "";
  • To find the length of a string, we use the built-in length property: str.length;
  • Strings and Numbers are Immutable that means we cannot change the state once it is declared.
  • However, we can assign the variable name to a new string..
  • Strings are case-sensitive, the lowercase and uppercase letters are treated as different values
  • We can also create strings using the new keyword.
