All about Arrays

  • Arrays are one of the most fundamental
  • Array is row in which data is stored and can be accessed easily
  • We can use the built-in method push() and unshift() to add elements to an array.
  • We can use the pop() method to remove the last element from an array,
    The pop() method also returns the returned value.
  • We can find the length of an element (the number of elements in an array) using the length property. ie arr.length
  • There are various methods we can use into arrays like forEach(), slice(), splice(), sort(), etc.
