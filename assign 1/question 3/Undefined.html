All about Null and Undefined

  • The value null represents the intentional absence of any object value. It is one of JavaScript's primitive values and is treated as falsy for boolean operations.
  • Difference between null and undefined
    null === undefined => false and
    null == undefined => true
  • Undefined is also a primitive value in JavaScript. A variable or an object has an undefined value when no value is assigned before using it.
  • We can assign null to a variable to denote that currently that variable does not have any value but it will have later on. A null means basicaly, absence of a value.
