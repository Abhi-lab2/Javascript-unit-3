All about Numbers

  • The Number constructor contains constants and methods for working with numbers. Values of other types can be converted to numbers using the Number() function.
  • A Number -
    Returns the value as a number.
    If the value cannot be converted to a number, NaN is returned.
    If no value is provided, 0 is returned.
  • The Number() function is used to convert data type to number.
  • There are various kinds of numbers like Decimal, Hexadecimal, Octal, Floating-point numbers, etc
  • For floating numbers we usually use Math.floor() method for complete value
  • For Ex. The .parseFloat() method parses a string value and returns the number with its decimal value.
